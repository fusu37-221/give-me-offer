# Role Playbooks

Use this file for role targeting, role-switching, and resume repositioning.

## Universal Role-Match Score

Score each target role from 0 to 100:

- 30: relevant experience or projects
- 20: hard skills/tools
- 15: domain knowledge
- 15: proof strength
- 10: communication/interview defensibility
- 10: short-term gap-closing feasibility

Return:

```text
Role:
Fit score:
Best evidence:
Main gaps:
Resume positioning:
One-week improvement:
Risk:
```

## Computer Science: Full-Stack to Frontend

Emphasize UI implementation, component reuse, Vue/React, state management, routing, forms, data tables, API integration, responsiveness, accessibility, performance, design collaboration, and product sense.

Downplay backend CRUD and database details unless they support frontend ownership.

Portfolio boost: rebuild one project frontend with polished UX, component notes, before/after screenshots, and performance checks.

## Computer Science: Full-Stack to AI Agent / AI Application

Emphasize API orchestration, workflow automation, tool calling concepts, data flow, prompt/RAG prototypes if real, evaluation logs if real, and business-process understanding.

Do not imply LLM platform expertise if the candidate only used ChatGPT casually.

Portfolio boost: build a small agent workflow for resume screening, document QA, customer-service triage, or campus assistant. Include README, architecture diagram, evaluation examples, and limitations.

## Non-CS to Product Assistant

Emphasize user research, requirement collection, competitive analysis, process mapping, data summary, cross-functional communication, writing, and presentation.

Portfolio boost: write a PRD for a familiar product and create a user journey map.

## Non-CS to Operations / New Media / E-Commerce

Emphasize content planning, user acquisition, community operation, event execution, data tracking, conversion funnel awareness, and copywriting.

Portfolio boost: produce a 7-day content calendar or analyze one account/shop campaign.

## Research Student to Industry R&D / Data / Product

Emphasize problem definition, literature review, experiment design, data processing, technical writing, collaboration, and reproducibility.

Translate papers and theses into projects only when the candidate can explain methods and contribution.
