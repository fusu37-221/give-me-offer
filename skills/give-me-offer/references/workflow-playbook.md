# Workflow Playbook

## Minimal Intake

Ask only what is needed to move:

1. Target role or JD.
2. Candidate background: education, work/internship/projects, skills.
3. Current resume if available.
4. Language and region: Chinese, English, bilingual, China, US, EU, etc.
5. Constraint: campus recruitment, internship, full-time, career switch, senior role.

If the user is brainstorming, produce strategy first and collect details later.

## Bullet Rewrite Formula

Use:

```text
Action + object/scope + method/tool + result/evidence
```

Before finishing, check that the target role is clear, strongest evidence appears early, bullets are defensible, unsupported metrics are marked or removed, and section order matches seniority and role.
