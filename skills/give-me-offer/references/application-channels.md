# Application Channels

Use this file when the user needs to adapt resume content to different delivery channels, not only a PDF resume.

## Purpose

The same candidate may need different materials for different channels. Keep the core facts consistent, but adapt length, tone, and evidence density.

## Channels

### China One-Page PDF

Use for:

- campus recruitment
- graduate recruitment
- most early-career roles
- ordinary social recruitment

Rules:

- A4 one page by default
- strongest role-relevant evidence first
- clean single-column layout
- PDF delivery plus editable source
- avoid long self-evaluation

### BOSS / Direct Chat Opening

Goal: get the recruiter to open the profile and continue the chat.

Structure:

```text
您好，我想应聘[岗位]。我有[最相关背景/项目/实习]，熟悉[2-3个岗位关键词]，最近做过[最强证明]。我看岗位重点是[JD重点]，这部分和我的经历比较匹配，想进一步了解机会。
```

Rules:

- 80-150 Chinese characters
- mention target role directly
- include one concrete proof
- avoid long biography
- avoid begging tone

### Online Application Fields

Common fields:

- project experience
- internship/work experience
- self-introduction
- reason for applying
- strengths/weaknesses
- career plan

Rules:

- reuse resume bullets but expand context slightly
- keep facts consistent with PDF
- avoid unsupported numbers
- answer the exact field prompt
- keep reusable field snippets in a source file if doing many applications

### Email Application

Use a concise email body:

```text
Subject: [Name] - Application for [Role] - [School/Current Role]

Dear [Name/Team],

I am applying for [role]. My background includes [1-2 strongest matching points]. I have attached my resume and [portfolio if relevant].

Thank you for your time.

Best regards,
[Name]
```

For China-facing email:

```text
邮件主题：[姓名]-应聘[岗位]-[学校/当前身份]

您好，

我想应聘贵司[岗位]。我的经历与岗位中的[关键词1/关键词2]较匹配，主要体现在[项目/实习/作品集一句话]。附件为我的简历，期待进一步沟通。

谢谢！
[姓名]
```

### LinkedIn / International Profile

Rules:

- no resume photo in the document unless region requires it
- headline should match target role
- About section should be concise and evidence-based
- use English action verbs
- keep portfolio/GitHub visible

### State-Owned Enterprise / Public Institution

Rules:

- stable, formal, conservative tone
- education, certificates, party/league status if relevant and true, awards, official records
- avoid internet slang and aggressive self-marketing
- include photo if the application convention expects it
- maintain detailed records for application forms, even if the resume remains one page

### Academic / Research CV

Use when:

- applying for academic programs
- research assistant roles
- PhD/postdoc
- roles requiring publication and project detail

Rules:

- two or more pages may be acceptable
- include publications, conference posters, patents, grants, datasets, methods
- distinguish personal contribution from team/lab project

## Consistency Check

Across all channels:

- job titles and dates match
- project names match
- numbers are consistent
- AI/technical claims use the same level
- no channel invents stronger ownership than the main resume
